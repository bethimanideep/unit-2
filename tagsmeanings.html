Tags Meaning

b

The b-tag specifies bold text without any extra importance.

i

The i HTML element represents a range of text that is set off from the normal text for some reason, such as idiomatic text

u

The u tag in HTML stands for underline, and it's used to underline the text

select

The select element is used to create a drop-down list.

button

The button tag in HTML is used to define the clickable button.

hr

The hr tag defines a thematic break in an HTML page and element is most often displayed as a horizontal rule that is used to separate content

br

The br tag inserts a single line break.
